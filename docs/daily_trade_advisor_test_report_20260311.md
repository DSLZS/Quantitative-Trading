## 5. Prompt 优化建议

基于本次测试结果，当前 SYSTEM_PROMPT 表现优秀，无需大幅修改。但可以提供以下微调建议：

### 5.1 现有 Prompt 优点

1. **关键词扫描完整**: 覆盖了立案调查、财务造假、大股东减持等核心风险
2. **小作文识别有效**: 能够区分官方公告与社交媒体传闻
3. **利好陷阱警惕**: 能够识别高送转掩护减持等虚假利好

### 5.2 可选增强方向

如需进一步提高审计锐度，可考虑：

1. **增加"问询函"关键词**: 交易所问询函通常预示潜在问题
2. **增加"诉讼仲裁"关键词**: 重大诉讼可能影响公司经营
3. **增加"业绩变脸"识别**: 警惕预告与实际差异过大的情况
