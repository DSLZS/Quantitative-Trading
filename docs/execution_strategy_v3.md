## 5 万元实盘最佳执行手册

### 核心问题解答

#### Q1: 到底应该选几只股？

**答：K = 3 只**

理由:
- 5 万元本金下，单股配置约 16667元
- K=3可在分散风险和避免碎股间取得平衡
- K 值过大导致单股金额过低，无法买足 100 股

#### Q2: 到底应该哪天调仓？

**答：每5天调仓一次**(周频)

理由:
- 日频调仓成本占比过高 (约 70%+)
- 5日调仓可降低交易次数，减少佣金侵蚀
- Alpha 信号在5日内衰减有限

#### Q3: 预测得分到多少才值得出手？

**答：概率阈值 >= 70%**

理由:
- 低于70%的股票胜率不足，不值得承担交易成本
- 高于70%的股票具有统计显著的 Alpha

### 最终配置建议

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 选股数量 (K) | 3 | 平衡分散与可执行性 |
| 调仓频率 | 5天 | 降低交易成本 |
| 概率阈值 | 70% | 确保胜率 |
| 排名退出 | 禁用 | 动态止损 |
| 国债 ETF | 启用 | 现金管理 |
