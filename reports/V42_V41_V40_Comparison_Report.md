## 六、V42 核心改进详解

### 6.1 信号净化：废除二阶动量

**V41 问题**:
```yaml
momentum_acceleration:
  权重：15%
  问题：产生过多假信号
  与现有因子相关性：0.78 (过高)
```

**V42 修复**:
```yaml
trend_quality (R²):
  权重：过滤器 (不直接加权)
  阈值：R² > 0.6 时动量信号才生效
  作用：过滤震荡市假信号
```

### 6.2 趋势质量因子 (R²) 计算

```python
def _compute_trend_quality(df: pl.DataFrame) -> pl.DataFrame:
    """
    V42 新增：趋势质量因子（线性回归 R²代理）
    
    计算过去 20 天价格序列的趋势清晰度
    R² > 0.6 表示趋势清晰，动量信号才生效
    """
    # 方向一致性 R²
    direction_r2 = 4.0 * (direction_consistency - 0.5).abs()
    
    # 趋势平滑度 R²
    trend_smoothness = 1.0 - price_std / (price_range + EPSILON)
    
    # 综合 R² = 0.5 * direction_r2 + 0.5 * trend_smoothness
    r2_combined = 0.5 * r2_proxy + 0.5 * trend_smoothness.clip(0, 1)
    
    return r2_combined
```

```python
def check_wash_sale(self, symbol: str, trade_date: str) -> Tuple[bool, Optional[str]]:
    """V42 洗售审计：检查是否在 5 天内卖出又买入"""
    # 检查今日是否已卖出
    if symbol in self.today_sells:
        return True, f"same_day_sell ({trade_date})"
    
    # 检查 5 天内是否有卖出记录
    if symbol in self.sell_history:
        last_sell_date = self.sell_history[symbol]
        days_diff = self._days_between(last_sell_date, trade_date)
        
        if days_diff <= WASH_SALE_WINDOW:  # 5 天
            return True, f"wash_sale_window ({days_diff} days)"
    
    return False, None
```
