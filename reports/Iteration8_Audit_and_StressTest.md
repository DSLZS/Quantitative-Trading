### 1.3 夏普计算逻辑复核

**修复前公式** (错误):
```python
sharpe = (mean_return - risk_free_rate) / std_return  # ❌ 未年化
```

**修复后公式** (正确):
```python
annualized_return = mean_return * 252
```

## 三、逻辑反思与修复说明

### 3.1 Bug 位置详情

#### Bug #1: T+0 成交 (未来函数)

**文件**: `src/backtest_engine.py` (旧版)

**问题描述**: 
在回测引擎中，T 日选出的股票使用 T 日收盘价成交。这在实际交易中是不可能的，因为：
- T 日收盘价只有在收盘后才能确定
- 实际交易中，T 日选股后最早只能 T+1 日开盘价成交

**修复方案**:
创建 `src/iteration8_stress_test.py`，实现 T+1 交易逻辑：
```python
# T 日选股
scored_stocks = self.compute_stock_scores(trade_date)

# T+1 日成交
next_date = self.get_next_date(trade_dates, trade_date)
next_data = self.get_stock_data(symbol, next_date)
buy_price = next_data["open"]  # 使用 T+1 开盘价
```

#### Bug #2: 夏普比率未年化

**文件**: `src/backtest_engine.py`

**问题描述**:
夏普比率公式分母使用日收益率标准差，未年化。

**修复方案**:
```python
# 修复前
sharpe = (mean_ret - risk_free) / std_ret

# 修复后
annualized_ret = mean_ret * 252
annualized_vol = std_ret * np.sqrt(252)
sharpe = (annualized_ret - risk_free) / annualized_vol
```

## 四、跨周期盲测结论

### 4.1 盲测设计

- **模型参数**: 保持不变 (使用原区间训练的简化动量模型)
- **测试区间**: 2024-05-01 ~ 2024-08-01 (模拟阴跌震荡市)
- **目的**: 验证策略在未见过的历史区间的表现

### 4.2 盲测结果

| 指标 | 原区间 A | 盲测区间 B | 差异 |
|------|---------|-----------|------|
| 总收益 | +8.92% | -3.43% | -12.35% |
| 夏普比率 | 1.03 | -0.43 | -1.46 |
| 最大回撤 | 16.24% | 19.20% | +2.96% |
| 胜率 | 55.93% | 48.39% | -7.54% |
| 超额收益 | -8.47% | +8.79% | +17.26% |

### 4.3 结论

1. **策略有效性**: 在阴跌震荡市 (区间 B) 中，策略虽然绝对收益为负 (-3.43%)，但**跑赢基准 8.79%**，显示了一定的抗跌性。

2. **市场适应性**: 
   - 区间 A (牛市): 策略收益 8.92% < 基准 17.39%，**跑输 8.47%**
   - 区间 B (熊市): 策略收益 -3.43% > 基准 -12.22%，**跑赢 8.79%**
   
   策略呈现**防御型特征**，在熊市中相对表现更好。

3. **风险控制**: ATR 动态止损在两个区间分别触发 17 次和 18 次，有效控制了单笔损失。
